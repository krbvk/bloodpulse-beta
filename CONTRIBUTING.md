### 1. Create a New Branch

Start by making sure you're on the develop branch and it's up to date:

```bash
git checkout develop
git pull origin develop
git checkout -b feature/your-feature-name
```

### 2. Make/Commit/Push 
```bash
git add .
git commit -m "Add: Login page UI with basic form validation"
git push origin feature/your-feature-name
```
